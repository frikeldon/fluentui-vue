# data-cell

## Props

| Prop name | Description                     | Type   | Values                    | Default |
| --------- | ------------------------------- | ------ | ------------------------- | ------- |
| content   | Text to render in the cell.     | string | -                         | ''      |
| align     | Alignment of the rendered text. | string | `left`, `center`, `right` | 'left'  |
