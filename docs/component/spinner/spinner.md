# spinner

## Props

| Prop name | Description                            | Type   | Values                               | Default  |
| --------- | -------------------------------------- | ------ | ------------------------------------ | -------- |
| size      | The size of Spinner to render.         | string | `large`, `medium`, `small`, `xSmall` | 'medium' |
| label     | The label to show next to the Spinner. | string | -                                    | ''       |
